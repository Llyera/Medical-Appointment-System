# Medical Appointment System

## COMP2090SEF / COMP8090SEF Course Project  

### Task 1: OOP-based Application
A complete clinic management system built with Python OOP concepts.

**Features**
- Patient management with medical records
- Doctor management
- Appointment booking system
- Medical history tracking

**OOP Concepts Used**
- Abstraction (Person abstract class)
- Inheritance (Patient & Doctor)
- Encapsulation (_pid)
- Polymorphism (display_info)
- Composition (Patient contains MedicalRecord)

**How to Run**
```bash
cd Task1_MedicalAppointmentSystem
python main.py
